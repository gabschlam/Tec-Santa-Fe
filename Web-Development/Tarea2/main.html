¿Qué es HTTP?

HTTP (“Hypertext Transfer Protocol”) es un protocolo que nos permite realizar una petición de datos y recursos, como, por ejemplo, documentos HTML. Es la base de cualquier intercambio de datos en la Web, utilizando una estructura Client – Server. Normalmente la petición de datos es iniciada por la entidad que recibirá los datos (Cliente).

¿Cómo es el protocolo?

Los clientes y servidores se comunican intercambiando mensajes. Los mensajes enviados por el cliente son peticiones y los mensajes enviados por el servidor son respuestas.

Historia

HTTP fue desarrollado por el World Wide Web Consortium y la Internet Engineering Task Force.
Comenzó a diseñarse y desarrollarse a principios de la década de 1990, el cual ha ido evolucionando a lo largo del tiempo, ya que HTTP es un protocolo ampliable.
Su primera versión se lanzó en 1991, la cual fue demasiado sencilla, que se volvió obsoleta en 1996, cuando salió la versión 1.0. Este proceso de cambio y búsqueda de una versión que cumpliera con el objetivo culminó en 1999, cuando se lanzó la versión 1.1, la cual es la versión más utilizada actualmente.
